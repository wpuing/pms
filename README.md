

## 1.简介

本系统为`简易小区物业管理系统`，是我大四上学期的J2EE`实训作业`，仅作为学习使用，综合自己学习过的一些知识写出来的。当然，项目功能还不够完善，可能存在很多问题，不喜勿喷。希望能得到一些建议。QQ865230476

**功能**

1. 管理员端：业主、员工、设施、收费、车位、房产、报修管理

2. 业主端：房产、车位信息查询，报修管理

3. 基础：登录、退出、修改密码、查看个人信息
